Most likely cause on Windows: another app (Zoom, Teams, Discord) has exclusive control of the microphone.

Fix options (try in order):
1. Close Zoom / Teams / Discord completely and click "Record 4 s" again.
2. Windows Settings -> System -> Sound -> Input -> choose your mic -> uncheck "Exclusive mode".
3. Use Firefox instead of Chrome - Firefox uses a different audio stack that bypasses exclusive mode.
4. Restart Chrome after closing other audio apps.
